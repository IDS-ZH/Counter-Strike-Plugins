# Реестр функционала автономных плагинов (Standalone)

## Immortal Hostages
- **Статус:** Исправлен и полностью работоспособен.
- **Описание:** Плагин управляет получаемым уроном для заложников. Отладочные логи (sm_hostages_debug "1") отрабатывают корректно, но могут сильно спамить в консоль. 
- **Режимы (sm_hostages_mode):**
  - `0` - Обычный режим (ванильный урон).
  - `1` - Уязвимы только для T (Террористов).
  - `2` - Уязвимы только для CT (Спецназа).
  - `3` - Полная неуязвимость.
- **Заметки:** Тестировался на карте `cs_assault` с включенным `mp_friendlyfire 1`. Режим `3` (invulnerable) подтвержден как 100% рабочий.

## Death Informer
- **Статус:** Протестирован и полностью работоспособен.
- **Описание:** Плагин информирует игрока о нанесенном и полученном уроне. 
- **Особенности:** Поддержка агрегации урона от дробовиков (`sm_deathinformer_shotgun_agg "1"`) работает идеально (суммирует урон от всех дробинок в один хит). Файл переводов `deathinformer.phrases.txt` обязательно должен находиться в папке `translations`.

## HUD_Damage (ShowDamage_MultiColour)
- **Статус:** Протестирован и полностью работоспособен.
- **Описание:** Плагин для вывода нанесенного урона на экран.
- **Особенности:** Вызывается в чате командами `!sd`, `!showdamage`, `!ыв`, `!ырщцвфьфпу`. Не генерирует никаких ошибок в логах SourceMod или движке.
