# Табличный обзор структуры проекта "Counter-Strike: Source Plugins"

## Структура основных директорий

| Директория | Тип | Содержание | Описание |
|------------|-----|------------|----------|
| CSS-GH | Исходный код | CSS_BASE-2007, CSS_SDK_2007 | Источники CSS 2007 года, SDK для разработки |
| ZH-IntegrationPack | Основной проект | Zh-sys, SRCDS-current-modded-catalog, xampp | Интеграционный пакет ZH-системы |
| Default_game_dir | Игровой каталог | cstrike, hl2 | Стандартные игровые директории |
| In Development/SourceMod/Legacy | Архив плагинов | Множество подкаталогов | Обширная коллекция исторических SourceMod плагинов |
| Documents | Документация | sourcemod-wiki, valve-srcds, mysql-refman | Документация и справочные материалы |

## Компоненты ZH-IntegrationPack

| Компонент | Тип | Назначение | Особенности |
|-----------|-----|------------|-------------|
| zh_core | Ядро | Основные нативы, логирование, пути к конфигам | Базовые функции системы |
| zh_modes | Ядро | Режимы DM/TDM/GG/Chicken/Revive | Переключение игровых режимов |
| zh_webbridge | Ядро | REST + WebSocket коммуникация | Удаленное управление сервером |
| zh_c4 | Ядро | Управление C4 | Система с кастомными временами детонации |
| zh_mst | Модификатор | Классы/модели/способности | ModelSwitchTool для скинов игроков/бота |
| zh_hostages | Модификатор | Управление заложниками | Настройка здоровья, маяки |
| zh_zones | Модификатор | Зоны на картах | Управление пользовательскими областями |
| zh_bots | Модификатор | Утилиты для ботов | Настройка сложности, подсветка |

## Архитектурные несогласованности

| Проблема | Описание | Влияние |
|----------|----------|---------|
| Классификация компонентов | ZH-MST функционирует как центральный компонент, но находится в модификаторах | Неопределенность в архитектуре |
| Несоответствие документации и реализации | Документация может не совпадать с реальной структурой | Осложнение разработки и сопровождения |
| Историческая эволюция | Проект включает элементы из разных фаз разработки | Потенциальные конфликты и дублирование функций |
